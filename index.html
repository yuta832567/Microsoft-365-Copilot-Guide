00

共通基盤と横断的な制約

7 ツールのブラウザ版で共通する前提条件・モデル選択・横断アップデートを整理します。各ツール固有の機能に入る前に、この章で基盤情報を把握してください。

01
Work IQ で組織データを横断参照
Microsoft 365 Copilot は Work IQ（ワーク・アイキュー）というインテリジェンス層を介して、メール・会議・ファイル・Teams メッセージをまたいだコンテキストを理解します。アクセス権限を尊重し、参照できるのは本人が権限を持つ情報のみです。
02
EDP でプロンプトは学習に使われない
エンタープライズデータ保護（EDP）が全ツールに適用されます。送信したプロンプトや応答は AI モデルの学習には使われず、Microsoft Product Terms のもとで保護されます。機密度ラベルも尊重されます。
03
Microsoft 365 Copilot アプリ刷新June
6 月から Microsoft 365 Copilot アプリの UI がチャット中心デザインに刷新中。チャットとレスポンスのレイアウトが整理され、ナビゲーション改善・作業整理用の新機能が追加されました。

用語補足 Key Terms

Core Concepts Edit with Copilot（旧称 Agent Mode） Word・Excel・PowerPoint のアプリ内部で Copilot が対話しながら直接編集する機能。2026 年 3 月の一般提供化時に「Agent Mode」から「Edit with Copilot」へ改称された。両方の名称が混在する。 Work IQ Copilot の背後にあるインテリジェンス層。ファイル・メール・会議の関係性を理解し、応答精度を高める。Microsoft 365 Copilot ライセンス保有者のみ利用可能。 Copilot Chat（グラウンディング） Copilot Chat 内部から各アプリの「Agent」（Word Agent / Excel Agent / PowerPoint Agent など）を呼び出せる機能。チャットから直接ドキュメントを生成し、OneDrive に保存される。 Frontier プログラム 一般提供（GA）前の機能を先行利用できる早期アクセスプログラム。管理者が Microsoft 365 管理センターで有効化する。 Copilot Notebooks ファイル・メール・会議・Web リンクなど複数のソースを集約して 1 つのワークスペースとして扱う機能。6 月から Word / PowerPoint / Excel への一括生成も可能に。

・入力制限：プロンプトの入力文字数および送信頻度には不正利用防止のための制限が設定されている

・生成質問数：1 フォームあたり Copilot は5〜15 問程度の質問を一括生成（トピックにより変動）

・回答数制限：Microsoft 365 サブスクライバーは 1 フォームあたり最大 50,000 回答まで収集可能（プランにより異なる）

・知識の鮮度：Copilot は学習データに基づくため、最新情報は正確でない場合あり（Web 検索は行わない）

・対応言語：複数言語対応。ただし言語により品質・サポート状況が異なる

・レビュー必須：生成されたドラフトは必ずユーザーがレビューしてから配信

・UX 課題：2026 年 6 月時点で、Copilot サジェスチョンをユーザー単位でオフにする設定は存在しない
