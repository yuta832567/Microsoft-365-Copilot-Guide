使い始める前に ― 3 つの前提

Before you start

① ライセンスが必要です

Microsoft 365 Business Standard と Microsoft 365 Copilot（アドオン）の両方が割り当てられている必要があります。画面右上のアカウントアイコン →「アカウントを表示」→「サブスクリプション」で確認できます。

② ファイルはクラウドに保存

Word・Excel・PowerPoint で Copilot をフルに使うには、ファイルが OneDrive（ワンドライブ）や SharePoint に保存されている必要があります。パソコンの中だけに保存したファイルでは一部機能が使えません。

③ 情報はきちんと守られます

入力した内容が AI の学習に使われることはありません（エンタープライズデータ保護）。また、Copilot が見られるのは、あなた自身が本来アクセスできる情報だけです。

Glossary
むずかしい言葉は、ここでチェック
Copilot コパイロット
Microsoft のアプリの中で働く AI アシスタント。文章の作成・要約、データの分析、スライド作りなどを手伝ってくれます。「副操縦士」という意味です。
Work IQ ワーク・アイキュー
Copilot の頭脳にあたる仕組み。あなたのメール・会議・ファイルのつながりを理解して、より的確な答えを返してくれます。
Copilot Chat コパイロット・チャット
チャット（会話）形式で AI に指示できる画面。ここから各ツールの「エージェント」を呼び出して、文書やスライドを作らせることもできます。
エージェント Agent
特定の仕事を任せられる AI の担当者。「Word エージェント」ならワード文書を、「Excel エージェント」なら表計算を作ってくれます。
Copilot Cowork コパイロット・コワーク
指示した仕事を「最後までやり切って完成品を返す」新しい AI。下書きや提案ではなく、複数アプリをまたぐ長い作業を裏側でこなします。2026 年 6 月 16 日に正式提供が始まりました（利用は管理者の設定が必要）。
プロンプト Prompt
AI に出す「指示文」のこと。「◯◯を作って」「これを要約して」のように、やってほしいことを言葉で伝えます。
OneDrive ワンドライブ
Microsoft のクラウド保存場所。ここにファイルを置くと、Copilot がその内容を読み取って作業を手伝えます。

どの AI モデルが使われる？

Available AI models

Copilot の裏側では、いくつかの AI モデル（頭脳の種類）が使い分けられています。用途に合わせて選べる場面もあります。

GPT-5.5 Instantジーピーティー・ファイブポイントファイブ・インスタント
応答がとても速いモデル。かんたんな要約や下書きなど、日常のほとんどの作業にちょうどよい。ふだんの基本モデル（既定）です。 Chat・Word・Excel・PowerPoint・Outlook・Teams
GPT-5.5 / 5.6 Thinking 7月ジーピーティー・シンキング
じっくり考えて答えるモデル。複雑な調べもの・分析向け。時間はかかりますが精度が高くなります。2026 年 7 月には、より新しい GPT-5.6 も Copilot に登場しました。 Copilot Chat・Researcher（リサーチャー）
Claude Opus 4.8 選択拡大クロード・オーパス・フォーポイントエイト
Anthropic（アンソロピック）社の最新モデル。多くの手順が必要な複雑な作業、長い文書やプレゼンの作成が得意。5 月 28 日から利用開始。6 月には Word でも選べるようになりました。 Copilot Chat・Word・Excel・PowerPoint・Copilot Cowork
Claude Sonnet 4.6 / Opus 4.7クロード・ソネット／オーパス
Anthropic 社のモデル。Copilot Cowork や調べもの、エージェント作成の場面で選べます。 Copilot Cowork・Researcher・Copilot Studio
モデルを選べるようにするには、会社の管理者が Anthropic を「副処理者（サブプロセッサー）」として有効にしておく必要があります（EU・EFTA・イギリスでは既定でオフ）。なお SharePoint の Copilot は GPT-5.4 Reasoning で動作し、モデル選択はできません。詳しくは社内の IT 担当者にご確認ください。

Word の Copilot

ワード ／ 文書の作成・編集
Word（ワード）の Copilot は、文章を一緒に書いてくれる AI パートナーです。「◯◯についての企画書を作って」とお願いすれば下書きを作り、「もっと簡潔に」と言えば書き直してくれます。長い文書の要約も得意です。2026 年 6 月からは Claude モデルの選択や画像生成も Word で使えるようになりました。
ブラウザでのアクセス先 word.cloud.microsoft 正式提供中

Word の画面でできること

In-app features
Edit with Copilot（エディット・ウィズ・コパイロット）旧 Agent Mode

指示文から文書を作り、会話しながら書き直せる機能。AI が「目的・読む人・トーン」を確認しながら完成度を高めます。2026 年 3 月以降、Web 検索や社内情報を使った部分には引用元が表示されるようになりました。

操作手順 ボタンの場所と手順を見る
  1. Word（ブラウザ版）で対象の文書を開く（OneDrive／SharePoint に保存されている必要があります）
  2. リボンの ホーム タブ右端の Copilot アイコンをクリック
  3. 右に開く Copilot 画面で、入力欄の下の Tools（ツール）アイコンをクリック
  4. メニューから Edit with Copilot を選ぶ
  5. 入力欄に指示を書いて送信
  6. 返ってきた文章の引用番号をクリックすると、参照元を確認できます
Draft with Copilot（下書き作成）

空白の行から、新しい文章をゼロから下書きしてもらう機能です。

操作手順 ボタンの場所と手順を見る
  1. 新しい段落の先頭にカーソルを置く
  2. 左の余白に出る Draft with Copilot アイコン（Copilot ロゴ）をクリック
  3. 指示を入力して Generate（生成）をクリック
  4. 結果を見て Keep it（採用）／Regenerate（作り直し）／Discard（破棄）を選ぶ
Rewrite（書き直し・短縮・トーン変更）

選んだ文章を書き直したり、短くしたり、丁寧な言い回しに変えたりできます。

操作手順 ボタンの場所と手順を見る
  1. 書き直したい文章を選択する
  2. 選んだ範囲の左余白に出る Copilot アイコンをクリック
  3. メニューから Rewrite（または「トーン変更」「短縮」）を選ぶ
  4. 提案を Replace（置き換え）で反映、または Try again（再生成）
Summarize（要約）

文書全体を要約し、要点と次にやることを整理してくれます。

操作手順 ボタンの場所と手順を見る
  1. リボンの ホーム タブ右端の Copilot アイコンをクリック
  2. 候補から Summarize this doc を選ぶ、または直接入力
  3. 要約の詳しさを「Brief／Standard／Detailed（簡潔／標準／詳細）」で切り替えられます
画像の生成・挿入6月

文書に合った画像を Copilot に作ってもらい、その場で挿入できます（Designer's Image Creator を使用）。

操作手順 ボタンの場所と手順を見る
  1. 画像を入れたい場所にカーソルを置く
  2. リボンの Copilot アイコンで Copilot ペインを開く
  3. 入力欄に作りたい画像を指示（例：「明るく自然光の入るモダンなオフィスの画像」）
  4. 生成された画像にマウスを重ね + をクリックして文書に追加
具体的に（形容詞・場所・「写実的」などの画風）書くほど狙いに近い画像になります。
Catchup（変更点のすばやい把握）6月

共有文書を前回開いてから何が変わったかを、カードでサッと要約してくれます。全部を読み返さずに最新状況をつかめます。

操作手順 ボタンの場所と手順を見る
  1. ほかの人が編集した共有文書を開く
  2. 上部に表示される Catchup カードをクリック
  3. 前回からの変更点の要約が表示されます
Lock-free coauthoring（同時編集の強化）5〜6月

複数人が同じ段落を同時に編集できるよう改良された共同編集機能。編集のぶつかり合いや待ち時間を減らします。

2026 年 5 月中旬〜6 月初旬に順次提供。うまく動かない状況では、これまでどおりの共同編集にそのまま戻ります。

できること・できないこと

Can / Cannot
できること
  • 指示文から本格的な長文の文書を作る
  • 会話しながら少しずつ書き直す・変更を確認する
  • 既存文書の要約・書き換え・トーン変更
  • メール・会議・ファイルから情報を引用して反映
  • 見出しや書式などのスタイルを保ったまま生成
  • Web 検索・社内情報の引用元をすぐ表示（3 月以降）
  • モデル選択（OpenAI／Claude）が Word でも可能に（6 月）
  • 文書内への画像生成・挿入（6 月）
  • Catchup：前回開いてからの変更点をすぐ把握（6 月）
  • コメントにもとづく編集・同じ段落の同時編集（6 月）
できないこと
  • 「抽出禁止」の機密ラベルがついた文書の処理
  • IRM 保護（情報の権限管理）がかかった文書の直接編集
  • オフラインでの利用（インターネット接続が必要）
  • 組織の外が持つ社外文書への参照
  • パソコンの中だけに保存したファイルでのフル機能利用
  • EU・EFTA・イギリスではClaude が既定オフ（管理者の有効化が必要）
使うときの注意点
  • ファイルの場所：OneDrive または SharePoint 上にある文書でのみフル機能が使えます
  • ファイルサイズ：Word から PowerPoint に変換するときは、元の Word 文書を 24 MB 以下にすると安定します
  • 共有ファイル：共有中の文書では、変更前に必ずプレビュー（確認画面）が表示されます
  • やり直し：Copilot の変更は履歴に残るので、いつでも前のバージョンに戻せます
  • モデル選択：2026 年 6 月から Word でも Claude を選べます。ただし EU・EFTA・イギリスでは既定オフのため、管理者による有効化が必要です

Excel の Copilot

エクセル ／ データの分析・集計
Excel（エクセル）の Copilot は、関数や専門知識がなくてもデータ分析ができるように手伝ってくれます。「顧客ごとの平均売上を出して」と言葉で頼むだけで数式を作り、グラフや集計表も提案します。5 月からは Plan モードや Python 分析、6 月からは スキル保存やブック単位のルールも加わり、より高度で一貫した作業ができるようになりました。
ブラウザでのアクセス先 excel.cloud.microsoft 正式提供＋大型更新

Excel の画面でできること

In-app features
Copilot を開く（共通の入口）

まずは Copilot 画面を開くところから。すべての機能はここから始まります。

操作手順 ボタンの場所と手順を見る
  1. OneDrive／SharePoint に保存したファイルを Excel（ブラウザ版）で開く
  2. リボンの ホーム タブ右端の Copilot アイコンをクリック
  3. 右に Copilot 画面が開くので、入力欄に指示を書く
Edit with Copilot（データ編集）

複数の手順が必要なデータ作業を、計画・実行してくれます。どんな手順で進めるかが目に見えるので安心。OpenAI と Claude のモデルを選べます。

操作手順 ボタンの場所と手順を見る
  1. Copilot 画面を開く（上の「Copilot を開く」参照）
  2. 入力欄の下の Tools アイコン（スライダー風の絵）をクリック
  3. メニューから Edit with Copilot（または Agent Mode）を選ぶ
  4. モデルを変えるときは、入力欄の下のモデル選択メニューから（OpenAI／Claude Opus 4.8 など）
  5. 指示を入力して送信。複雑な依頼は数分かかることがあります
  6. 結果を確認し、必要なら Ctrl＋Z で取り消し
Plan モード（先に手順を見せてくれる）5月

編集する前に、Copilot が「どのデータを使い、どんな順番で作業するか」を一覧で見せてくれます。内容を確認してから実行できるので、思わぬ変更を防げます。

操作手順 ボタンの場所と手順を見る
  1. Copilot 画面を開く
  2. 入力欄の上のメニューから Plan（プラン）を選ぶ
  3. 指示を入力（例：「月ごとの推移グラフと前年比を追加」）
  4. Copilot が作業の手順を一覧で提示
  5. 必要なら手順を調整して Apply（適用）で実行
Python（パイソン）による高度分析5月

Copilot が Python というプログラミング言語を裏側で使い、より本格的なデータ分析やグラフ作成を行います。コードを書く必要はありません。

操作手順 ボタンの場所と手順を見る
  1. Edit with Copilot を開く
  2. 指示に「Python を使って」と書くか、高度な分析を頼むと Copilot が Python を呼び出します
  3. 例：「Python を使って、この売上データの季節性を分析し、来期予測グラフを作って」
  4. 作られた分析結果とグラフがシートに挿入されます
会社の管理者が「Python in Excel」を許可している必要があります。
数式を言葉で作る

「顧客ごとの平均売上を計算する列を追加」のように言葉で頼むだけで、数式を作ってくれます。

操作手順 ボタンの場所と手順を見る
  1. 対象のデータ範囲をテーブル化しておく（Ctrl＋T）と精度が上がります
  2. Copilot 画面に「◯◯を計算する列を追加」と入力
  3. 提示された数式のプレビューを確認して Insert column（列を挿入）
=COPILOT() 関数（セルの中で AI を使う）

セルの中でプロンプトを実行できる新しい関数。参照先のセルが変わると、結果も更新されます。

操作手順 ボタンの場所と手順を見る
  1. 任意のセルに =COPILOT("指示文", 参照範囲) と入力
  2. 例：=COPILOT("このレビューの感情は？", A2) で口コミの感情を分析
  3. 参照セルが変わると結果も再計算されます
スキル（よく使う指示を保存）6月

分析・モデリング・レポート作成など、繰り返す作業を「スキル」として保存し、次回から呼び出すだけで実行できます。同じ指示を毎回書く手間がなくなります。

2026 年 6 月から順次提供。まずは Insider（先行利用）向けから始まっています。
個人設定（あなたの好みを覚える）6月

書式・命名ルール・数式の書き方・ピボットやレポートのスタイルなど、自分の好みを一度設定すると、どのブックでも Copilot が同じように反映してくれます。

ブックのルール（.Rules シート）6月

特定のブックにだけ適用したいルール（構造・書式・命名・数式の決まりなど）を .Rules という専用シートに書いておくと、そのファイルを編集する全員の Copilot が同じルールに従います。テンプレートの標準化に便利です。

Teams 管理センターで Facilitator アプリと Loop コンポーネントの有効化が前提です。Web 検索による回答は、既定でオフ・管理者が有効化した場合のみ動作します。

話者が話し終えてから翻訳する新しい通訳モードです。通訳役（Interpreter エージェント）が会議画面に登場します。

操作手順 ボタンの場所と手順を見る

  2. メニューで Consecutive interpretation を選ぶ
  3. 翻訳する 2 言語のペアを選んで有効化
  4. 通訳役が会議画面に登場し、全員に共有されます
対応言語（10 種）：英語・フランス語・ドイツ語・スペイン語・イタリア語・ポルトガル語・中国語（簡体字／繁体字）・日本語・韓国語。月 20 時間／人まで。主催者のみライセンスが必要（参加者は不要）です。
AI のオン／オフを会議中に切替7月

会議の主催者・発表者は、会議中でも Copilot・Facilitator・Recap（要約）のオン／オフを切り替えられます。個別に切り替えることも、まとめて切り替えることもできます。機密の話をするときに便利です（7 月に順次正式提供）。

操作手順 ボタンの場所と手順を見る

  2. Copilot・Facilitator・Recap のトグルで個別に切り替え
AI を完全にオフにするには、文字起こしもオフにする必要があります（両者は連動しています）。過去の会議には影響しません。
